# LuluPet v1.2.0

v1.2.0 adds interaction controls for desktop movement and improves clipboard history reliability on top of v1.1.1.

## What's New Since v1.1.1

- Added `Walk Mode` as a user-controlled option in both the tray menu and the settings panel.
  - When Walk Mode is off, LuluPet stays at the current position instead of randomly walking around.
  - When Walk Mode is on, the existing random walk behavior is enabled.
- Reminder screen laps remain independent from Walk Mode.
  - Pomodoro, water, and stand reminders can still trigger the existing one-lap walk around the current screen.
  - Clicking or dragging LuluPet stops the active reminder lap and returns the pet to `Idle`.
- Clipboard history monitoring now runs continuously after startup.
  - The existing Win32 clipboard notification path is preserved.
  - A low-frequency polling fallback keeps clipboard history working even if system clipboard notifications fail.

## Configuration

- Added `interaction.walkMode`.
  - Default: `false`.
  - Existing settings files without this key continue to load with the default value.
