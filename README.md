# LuluPet

LuluPet is a Windows-only WPF desktop pet application.

## Development

The UI project targets `net8.0-windows` and is built on GitHub Actions with a
Windows runner. Core and platform wrapper projects can be checked from Ubuntu
where possible.

## Run

Publish or build `src/LuluPet.App/LuluPet.App.csproj` on Windows, or download
the `LuluPet-win-x64` artifact generated by GitHub Actions.

For the P4 animation player, place PNG frames under:

```text
assets/pet/idle/lulu_idle_0001.png
assets/pet/walk/lulu_walk_0001.png
assets/pet/sleep/lulu_sleep_0001.png
```

At runtime, `I`, `W`, and `S` switch between Idle, Walk, and Sleep for manual
acceptance checks.

## Settings

Use the tray menu item `设置` to change runtime settings. Changes are applied
immediately and persisted to `settings.json`.

Supported settings:

- `appearance.scale`: pet window scale, clamped to `0.75`-`1.5`
- `appearance.opacity`: pet window opacity, clamped to `0.35`-`1.0`
- `audio.volume`: reserved volume setting, clamped to `0.0`-`1.0`
- `interaction.clickThrough`: enables click-through window style
- `startup.autoStart`: writes/removes the current-user Windows `Run` value

SQLite runtime data is stored under:

```text
%LOCALAPPDATA%\LuluPet\data\lulupet.db
```
