# LuluPet v1.1.0

v1.1.0 focuses on desktop workflow tools and panel polish on top of v1.0.3.

## What's New Since v1.0.3

- Added clipboard history panel with `C`.
  - Keeps the latest 100 text entries.
  - Click an entry to write it back to the system clipboard.
  - Clipboard history is now saved locally and restored after restart.
- Added file transit panel with `D`.
  - Drag files into the panel to temporarily store copies.
  - Press `Ctrl+V` in the panel to store copied files or an unsaved clipboard image.
  - Double-click a stored file to open it.
  - Right-click a stored file to copy it for Explorer paste.
  - Custom transit folder can be set from Settings.
- Added tray Tutorial panel.
  - Covers `F`, `C`, `D`, `W`, `I`, `S`, clipboard behavior, and file transit behavior.
- Reworked Settings into the same bubble panel style as `F`/`C`/`D`.
- Improved animation smoothness.
  - Animation ticks now use WPF rendering cadence.
  - Pet frames are preloaded and frozen before playback.
  - Image scaling is optimized for smoother frame switching.

## Fixes

- Fixed Win11 file copy behavior from the file transit panel by using Explorer-friendly clipboard file-drop data.
- Fixed screenshot paste in file transit so clipboard images are saved as valid PNG files.
- Fixed initial tray menu state so `Hide` is available when LuluPet is visible at startup.
- Preserved click-through behavior: tool panel hotkeys only work when click-through is off.
