# LuluPet v1.1.1

v1.1.1 is a patch release for animation interaction polish and desktop movement stability on top of v1.1.0.

## What's New Since v1.1.0

- Single-click pet interaction now randomly plays `Happy` or `Surprised`.
  - If one animation set is missing, LuluPet falls back to the other one.
  - If both are missing, it falls back to `Idle`.
- Pet animation assets have been normalized to continuous frame names from `0001` in each action folder.
  - `happy`: `0001` to `0020`.
  - `walk`: `0001` to `0006`.
  - Other action folders remain continuous.

## Fixes

- Fixed flickering while changing the pet scale.
- Fixed flickering when dragging the pet against a screen edge and continuing to drag.
- Preserved the existing walk-facing mirror behavior, so walk direction still follows horizontal movement.

- `LuluPet-win-x64-v1.1.1.zip`: Windows x64 package including `LuluPet.exe`, `Assets`, and `settings.json`.
