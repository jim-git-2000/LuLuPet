# LuluPet

LuluPet is a Windows-only WPF desktop pet application.

## Development

The UI project targets `net8.0-windows` and is built on GitHub Actions with a
Windows runner. Core and platform wrapper projects can be checked from Ubuntu
where possible.

## Run

Publish or build `src/LuluPet.App/LuluPet.App.csproj` on Windows, or download
the `LuluPet-win-x64` artifact generated by GitHub Actions.

For the P4 animation player, place PNG frames under:

```text
assets/pet/idle/lulu_idle_0001.png
assets/pet/walk/lulu_walk_0001.png
assets/pet/sleep/lulu_sleep_0001.png
```

At runtime, `I`, `W`, and `S` switch between Idle, Walk, and Sleep for manual
acceptance checks.
